- [ ]  **4. 회고를 잘 작성했나요?**
    - 주어진 문제를 해결하는 완성된 코드 내지 프로젝트 결과물에 대해
    배운점과 아쉬운점, 느낀점 등이 기록되어 있는지 확인
    - 작은 데이터셋에서의 과적합을 정확히 파악 후 원인을 분석하여 단순히 정확도가 낮다에서 그치지 않고 짚어내 작성하여주셨습니다.
